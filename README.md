# go-postgres

PostgreSQLデータベース管理システムのGoによる実装です。

## About

このプロジェクトは、PostgreSQLのコアコンポーネントをGoで再実装する試みです。
オリジナルのPostgreSQLの設計と実装を参考にしています。
